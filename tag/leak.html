故事是這樣的

仔細一查發現是記憶體使用過多，因為有別人的程式需要資源，跑到一半就被 OS 給砍了

（題外話，砍的優先順序包括你的程式跑了多久、吃多少記憶體等等一堆參數）

而我心中以為，某些用不到的 object reference 在 function return 之後就會釋放

顯然並沒有

但實驗證實也完全沒有用
